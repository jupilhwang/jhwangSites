# PowerShell 
- Windows에서 만든 Shell로 OSS로 모든 OS에서 사용가능한 Shell이다. bash 나 다른 shell 에 비해서 많이 사용하지 않지만, 모든 OS에서 동작하는 스크립트를 만든다고 하면 좋은 대안이다.

- vSphere의 Automation 을 모듈인 PowerCLI는 PowerShell 기반으로 만들어져 있다.

- PowerShell 설치

  - Linux

  - MacOS
